# Target profile — onco-immunology drug-target program

A drug-target program deck covering genetic / functional-genomic evidence, clinical
feasibility, novel-target strategy, a development plan, and spatially-rationalized
bispecific antibody designs for a tumor immunosuppressive axis.

## Contents
- **`target_profile.pdf`** — 10-page PDF (cover, executive summary, figure pages, grounding).
- **`index.html`** — reveal.js web deck (same narrative). Serve via GitHub Pages or open locally.
- **`figures/`** — source figures (PNG).

## Grounding
Drug / ligand data: ChEMBL · Trials: ClinicalTrials.gov + PubMed · Evidence: Open Targets ·
Networks: STRING · Expression / compartments: Human Protein Atlas + CELLxGENE CellGuide ·
Spatial adjacency: peer-reviewed studies. Patents: keyword-level web scan (not a professional
FTO opinion). Spatial adjacency was read from published figures/text, not recomputed from raw spots.
